**Таблица 2.1** — Модель учета техники и формирование QR-кода  
Файл: `webapp/backend/apps/assets/models.py`

```python
class Asset(models.Model):
    STATUS_CHOICES = [
        ('warehouse', 'На складе'), ('in_use', 'В эксплуатации'),
        ('repair', 'В ремонте'), ('written_off', 'Списан'),
    ]
    inventory_number = models.CharField(max_length=50, unique=True)
    name = models.CharField(max_length=255)
    status = models.CharField(max_length=20, choices=STATUS_CHOICES)
    photo = models.ImageField(upload_to='assets/photos/', null=True, blank=True)
    qr_code = models.ImageField(upload_to='assets/qr/', null=True, blank=True)

    def generate_qr_code(self):
        qr = qrcode.QRCode(version=1, box_size=8, border=2)
        qr.add_data(f'IT-ASSET:{self.inventory_number}:{self.name}')
        qr.make(fit=True)
        buffer = io.BytesIO()
        qr.make_image().save(buffer, format='PNG')
        self.qr_code.save(f'qr_{self.inventory_number}.png',
                          ContentFile(buffer.getvalue()), save=False)
```

*Назначение:* хранение данных об оборудовании, статусы жизненного цикла, QR-код при регистрации.

**Таблица 2.2** — Массовое создание техники с серией инвентарных номеров  
Файл: `webapp/backend/apps/assets/services.py`

```python
@transaction.atomic
def create_assets_bulk(validated_data: dict, quantity: int) -> list[Asset]:
    numbers = generate_inventory_numbers(
        validated_data['inventory_number'].strip(), quantity)
    if Asset.objects.filter(inventory_number__in=numbers).exists():
        raise ValueError('Инвентарные номера уже заняты')
    created = []
    for index, inv in enumerate(numbers):
        row = {**validated_data, 'inventory_number': inv}
        if index > 0:
            row.pop('photo', None)
        created.append(Asset.objects.create(**row))
    return created
```

*Назначение:* поступление партии однотипной техники — несколько карточек с последовательными номерами.

**Таблица 2.3** — Ремонтная заявка и расчет времени простоя  
Файл: `webapp/backend/apps/maintenance/views.py`, `models.py`

```python
def perform_create(self, serializer):
    ticket = serializer.save(reported_by=self.request.user)
    asset = ticket.asset
    if asset.status != Asset.STATUS_REPAIR:
        asset.status = Asset.STATUS_REPAIR
        asset.save(update_fields=['status'])

@property
def downtime_hours(self):
    end = self.resolved_at or timezone.now()
    return round((end - self.created_at).total_seconds() / 3600, 1)
```

*Назначение:* регистрация заявки, перевод техники в ремонт, учет длительности простоя в часах.

**Таблица 2.4** — Замена неисправной техники единицей со склада  
Файл: `webapp/backend/apps/maintenance/views.py`

```python
@transaction.atomic
def replace_from_warehouse(self, request, pk=None):
    ticket, broken = self.get_object(), self.get_object().asset
    replacement = Asset.objects.get(pk=request.data['replacement_asset_id'])
    replacement.location = broken.location
    replacement.responsible_employee = broken.responsible_employee
    replacement.status = Asset.STATUS_IN_USE
    replacement.save()
    broken.status = Asset.STATUS_WAREHOUSE
    broken.save()
    ticket.asset = replacement
    ticket.save(update_fields=['asset'])
```

*Назначение:* выдача исправной техники пользователю, неисправная единица возвращается на склад.
